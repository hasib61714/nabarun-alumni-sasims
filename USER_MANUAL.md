## ২. ছাত্র নিবন্ধন ও লগইন

### নিবন্ধন করুন

1. সাইটে যান → উপরে **"রেজিস্টার"** বাটন চাপুন অথবা `/register` এ যান
2. পূরণ করুন:
   - **পুরো নাম** — বাংলা বা ইংরেজিতে
   - **ইমেইল** — আপনার সক্রিয় ইমেইল
   - **SSC ব্যাচ বছর** — ড্রপডাউন থেকে বেছে নিন
   - **পাসওয়ার্ড** — কমপক্ষে ৬ অক্ষর
   - **পাসওয়ার্ড নিশ্চিত করুন** — একই পাসওয়ার্ড আবার দিন
3. **"রেজিস্টার করুন"** বাটন চাপুন
4. সফল হলে সবুজ সাফল্য স্ক্রিন দেখাবে
5. ইমেইল ভেরিফিকেশন লিংক আসবে — ক্লিক করুন

> ⚠️ রেজিস্ট্রেশনের পর **অ্যাডমিন** আপনার প্রোফাইল অনুমোদন না করা পর্যন্ত ডিরেক্টরিতে দেখা যাবে না।

### লগইন করুন

1. `/login` এ যান
2. **"ছাত্র লগইন"** ট্যাব সিলেক্ট করুন (ডিফল্ট)
3. ইমেইল ও পাসওয়ার্ড দিন
4. **"লগইন করুন"** চাপুন → ছাত্র ড্যাশবোর্ডে যাবেন

### পাসওয়ার্ড ভুলে গেলে

1. লগইন পেজে **"পাসওয়ার্ড ভুলে গেছেন?"** চাপুন
2. ইমেইল দিন → **"রিসেট লিংক পাঠান"** চাপুন
3. ইমেইলে আসা লিংকে ক্লিক করে নতুন পাসওয়ার্ড সেট করুন

## ৩. ছাত্র ড্যাশবোর্ড

লগইনের পর `/dashboard` এ স্বয়ংক্রিয়ভাবে যাবেন।

### প্রোফাইল সম্পাদনা

ড্যাশবোর্ডে যা যা আপডেট করতে পারবেন:

| বিভাগ | তথ্য |
| --- | --- |
| **ব্যক্তিগত তথ্য** | নাম, রোল নম্বর, ব্যাচ, ছবি আপলোড |
| **একাডেমিক ফলাফল** | SSC ফলাফল (GPA/গ্রেড) |
| **অর্জনসমূহ** | পুরস্কার, বিশেষ স্বীকৃতি |
| **পেশাগত তথ্য** | বর্তমান পেশা, উচ্চশিক্ষা |
| **সোশ্যাল লিংক** | LinkedIn, Facebook প্রোফাইল লিংক |

**ছবি আপলোড:**

1. প্রোফাইল সেকশনে **"ছবি পরিবর্তন"** চাপুন
2. JPG/PNG ফাইল সিলেক্ট করুন
3. আপলোড সম্পন্ন হলে সংরক্ষণ করুন

### সাম্প্রতিক নোটিশ

ড্যাশবোর্ডের উপরে সাম্প্রতিক নোটিশগুলো দেখা যাবে।

### মতামত (Testimonial)

নিচের **"আপনার মতামত"** সেকশনে সাফল্যের গল্প বা মতামত লিখে জমা দিতে পারবেন — অ্যাডমিন অনুমোদন করলে হোম পেজে দেখাবে।

## ৪. অ্যাডমিন লগইন

1. `/login` এ যান
2. **"অ্যাডমিন লগইন"** ট্যাব সিলেক্ট করুন
3. অ্যাডমিন ইমেইল ও পাসওয়ার্ড দিন
4. লগইন করুন → অ্যাডমিন ড্যাশবোর্ডে যাবেন

> ⚠️ অ্যাডমিন ট্যাবে ছাত্রের account দিয়ে লগইন করলে সাথে সাথে লগআউট হয়ে ত্রুটি বার্তা দেখাবে।

## ৫. অ্যাডমিন ড্যাশবোর্ড

অ্যাডমিন ড্যাশবোর্ডে ৮টি ট্যাব আছে:

### ট্যাব ১ — ছাত্র ব্যবস্থাপনা

- সব নিবন্ধিত ছাত্রের তালিকা দেখুন
- **অনুমোদন / প্রত্যাখ্যান** করুন (সবুজ ✓ / লাল ✗ বাটন)
- **বাল্ক অনুমোদন:** একাধিক ছাত্র চেকবক্স দিয়ে সিলেক্ট করে একসাথে অনুমোদন/প্রত্যাহার
- **সার্চ:** নাম বা রোল দিয়ে খুঁজুন
- **ফিল্টার:** ব্যাচ বছর বা অনুমোদন স্ট্যাটাস অনুযায়ী
- **বিস্তারিত দেখুন:** যেকোনো ছাত্রের নামে ক্লিক করুন
- **ছাত্র মুছুন:** লাল Trash আইকন চাপুন
- **CSV এক্সপোর্ট:** "ডাউনলোড CSV" বাটন দিয়ে সব ছাত্রের তথ্য Excel-এ নামিয়ে নিন

### ট্যাব ২ — গ্যালারি

- বিদ্যমান ছবি দেখুন ও প্রকাশ/গোপন করুন (Toggle)
- **নতুন ছবি আপলোড:**
  1. শিরোনাম দিন
  2. ইভেন্টের নাম ও ব্যাচ বছর (ঐচ্ছিক)
  3. ছবি ফাইল সিলেক্ট করুন
  4. **"আপলোড করুন"** চাপুন
- ছবি মুছতে Trash আইকন চাপুন

### ট্যাব ৩ — ইভেন্ট

- চলমান ইভেন্টের তালিকা ও প্রকাশ/গোপন toggle
- **নতুন ইভেন্ট:**
  1. শিরোনাম, বিবরণ দিন
  2. তারিখ ও সময় সেট করুন
  3. স্থান লিখুন
  4. বিভাগ বেছে নিন
  5. **"ইভেন্ট যোগ করুন"** চাপুন

### ট্যাব ৪ — অনুদান

- সব অনুদান রেকর্ড দেখুন
- অনুদান **অনুমোদন / প্রত্যাখ্যান** করুন
- **নতুন ক্যাম্পেইন:**
  1. শিরোনাম ও বিবরণ দিন
  2. লক্ষ্যমাত্রা (টাকা) দিন
  3. **"ক্যাম্পেইন তৈরি করুন"** চাপুন
- ক্যাম্পেইন সক্রিয়/নিষ্ক্রিয় toggle করুন

### ট্যাব ৫ — পরিসংখ্যান

- ব্যাচ অনুযায়ী ছাত্র সংখ্যার বার চার্ট
- অনুমোদিত vs অপেক্ষমান পাই চার্ট
- পেশা অনুযায়ী বিতরণ চার্ট

### ট্যাব ৬ — বার্তা (Inbox)

- যোগাযোগ ফর্ম থেকে আসা সব বার্তা
- অপঠিত বার্তায় লাল badge দেখাবে
- বার্তা ক্লিক করলে বিস্তারিত দেখা যাবে ও পঠিত চিহ্নিত হবে
- বার্তা মুছতে Trash আইকন চাপুন

### ট্যাব ৭ — নোটিশ

- প্রকাশিত নোটিশের তালিকা ও প্রকাশ/গোপন toggle
- **নতুন নোটিশ:**
  1. শিরোনাম দিন
  2. বিষয়বস্তু লিখুন
  3. **"নোটিশ যোগ করুন"** চাপুন

### ট্যাব ৮ — টেস্টিমোনিয়াল

- ছাত্রদের জমা দেওয়া মতামত
- **অনুমোদন** করলে হোম পেজে দেখাবে
- **প্রত্যাহার** বা **মুছুন** করতে পারবেন

## ৬. সুপার অ্যাডমিন

সুপার অ্যাডমিন লগইনের পর `/super-admin` পেজে যাবেন।

### সুপার অ্যাডমিন ড্যাশবোর্ড ফিচার

- **মোট ব্যবহারকারী** — সিস্টেমের সব user-এর সংখ্যা
- **ভূমিকা ফিল্টার ট্যাব:** সবাই / অ্যাডমিন / সুপার অ্যাডমিন / ছাত্র
- **নাম দিয়ে সার্চ**

### অ্যাডমিন নিয়োগ দেওয়া

1. ছাত্রের নাম খুঁজুন
2. **"অ্যাডমিন করুন"** বাটন চাপুন (সবুজ Shield আইকন)
3. নিশ্চিতকরণ ডায়ালগে **"নিশ্চিত করুন"** চাপুন
4. সেই ছাত্র এখন অ্যাডমিন লগইন করতে পারবে

### অ্যাডমিন রোল সরানো

1. অ্যাডমিনের নাম খুঁজুন
2. **"রোল সরান"** বাটন চাপুন (লাল আইকন)
3. নিশ্চিতকরণ ডায়ালগে **"নিশ্চিত করুন"** চাপুন

### অ্যাডমিন ড্যাশবোর্ড থেকে সুপার প্যানেলে যাওয়া

অ্যাডমিন ড্যাশবোর্ডের হেডারে (super_admin হলে) **"সুপার প্যানেল"** বাটন দেখাবে।

## ৭. প্রথমবার সিস্টেম সেটআপ গাইড

প্রথমবার সিস্টেম চালু করার ধাপ:

### ধাপ ১ — Migration চালান

Supabase Dashboard → SQL Editor এ [supabase/migrations/20260305000000_add_super_admin_role.sql](supabase/migrations/20260305000000_add_super_admin_role.sql) ফাইলের সম্পূর্ণ SQL paste করে Run করুন।

### ধাপ ২ — সুপার অ্যাডমিন রেজিস্টার করুন

`/register` পেজে গিয়ে আপনার ইমেইল/পাসওয়ার্ড দিয়ে একটি account তৈরি করুন।

### ধাপ ৩ — User ID বের করুন

Supabase → Authentication → Users → আপনার ইমেইলের পাশের UUID কপি করুন।

### ধাপ ৪ — সুপার অ্যাডমিন রোল দিন

Supabase SQL Editor এ চালান:

```sql
INSERT INTO public.user_roles (user_id, role)
VALUES ('আপনার-UUID-এখানে', 'super_admin');
```

### ধাপ ৫ — লগইন করুন

`/login` → **"অ্যাডমিন লগইন"** ট্যাব → ইমেইল/পাসওয়ার্ড → Super Admin Dashboard।

### ধাপ ৬ — অন্যদের অ্যাডমিন বানান

Super Admin Dashboard থেকে যে ছাত্রকে অ্যাডমিন করতে চান তাকে **"অ্যাডমিন করুন"** বাটনে ক্লিক করুন।
